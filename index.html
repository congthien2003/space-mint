Features

Built for developers who juggle projects

Everything you need to organize, run, and switch between work — without the clutter.

Add Project

Drop in any local repo or folder. Projects are indexed and ready to run in seconds.

Workspaces

Group related projects into a workspace. Switch context instantly without losing your setup.

Multiple Terminals

Run as many terminals as you need — each tied to its own project, all in one window.

Split Terminal

Divide a pane into splits to watch logs and run commands side by side, in real time.
